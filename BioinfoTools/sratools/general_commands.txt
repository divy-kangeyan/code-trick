SRA(Sequence Read Archive)
——————————————————
sra-tools can be used to perform analysis related to sra files

sra-tools help you to access the data housed in SRA and convert them into fastq/fasta/bam 
or other types of sequencing files

sra-tools can also convert any type of sequencing file to and sra file


* Download the SRA files given an SRA ID


* Download the fastq file given an SRA ID

* Download an aligned file

sam-dump SRA_id > SRA_id.sam
